Introducción a TypeScript

¿Que es TypeScript?

  • Lenguaje de programación tipado
  • Open Source
  • Desarrollado por Microsoft
  • Lenguaje de programación oficial de google
  • Podemos usar las últimas características de JavaScript
  • Este codigo de alto nivel necesita ser compilado
  • El proceso de compilación genera código JavaScript
  • Podemos ejecutar el código JavaScript en cualquier navegador o Sistema Operativo

¿Por qué usar TypeScript?

  • Orientado a Objetos
  • Soporte de verificación de tipos estáticos
  • Soporte de ES5, ES6, más
  • Se actualiza periódicamente
  • Comunidad activa
  • Puede prevenir cerca del 15% de bugs

El compilador TypeScript


        
            /**********************************************************************/
            node --version
            //resultado
            v12.16.1
            /**********************************************************************/
            npm install -g typescript
            tsc --version
            //resultado
            3.8.3
            /**********************************************************************/
            //para compilar un archivo typescript se ejecuta el siguiente comando
            tsc src/hola.js

El archivo tsconfig


        
            /**********************************************************************/
            //creando el archivo tsconfig.json
            tsc --init
            //el parametro [RutaArchivo] es opcional
            tsc --watch [RutaArchivo]

El compilador y tsconfig


        
            /**********************************************************************/
            {
                "compilerOptions":{
                    ...
                },
                "include":{
                    "src/*.ts",
                    "src/**/*.ts"
                },
                "exclude":{
                    "src/test.ts",
                    "src/*-test.ts"
                }
            }

El compilador y ECMASript


        
            /**********************************************************************/
            //Esta opción sirve para redireccionar los archivos generados por el compilador
            //a la carpeta personalizada
            {
                "compilerOptions": {
                    "outDir": "./dist"
                }
            }
